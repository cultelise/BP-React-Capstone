# Elise-Closet

## Features
* Displays all garments
* Search by name, style, or color
* Specific garment page with additional details
* Garment uploading with multi-image option
 

## Technologies
### Front End
* React
* React-Router
* CSS
* JavaScript
* Axios

### Back End
* NodeJS
* Express
* GoogleAPIs
* Sequelize
